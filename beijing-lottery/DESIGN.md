# 北京小客车摇号模拟器 — 设计文档

## 一、业务背景

模拟北京燃油车个人小客车指标摇号。北京自 2011 年起实行摇号制度，2021 年改革为：

- 燃油车指标 **每年两期**（6 月、12 月各一次）
- 个人参与摇号，**阶梯基数递增**（摇得越久，权重越大）
- 新能源车为轮候排队制（本模拟器暂不覆盖）
- 2021 年后新增家庭摇号（本模拟器暂仅模拟个人）

## 二、核心规则

### 2.1 池子参数

| 参数 | 值 | 说明 |
|------|-----|------|
| POOL_SIZE | 2,600,000 | 个人燃油车申请人数（2024-2025 约 260 万） |
| QUOTA_PER_DRAW | 19,000 | 每期分配指标数（全年约 38,000） |
| DRAWS_PER_YEAR | 2 | 一年两期（6月 + 12月） |
| CHANCE_INTERVAL | 2 | 每 2 次不中基数 +1 |
| MIN_CHANCES | 1 | 初始基数 |
| AVG_OTHERS_DRAWS | 12 | 全池平均摇号次数（约 6 年） |

### 2.2 阶梯基数公式

```
基数 = 1 + floor(总摇号次数 / 2)
```

| 摇号次数 | 基数 | 说明 |
|---------|------|------|
| 0 | 1 | 刚申请 |
| 1-2 | 1 | 第一次摇、第二次摇（仍算 2 次不中才加） |
| 3-4 | 2 | 约 1-2 年 |
| 5-6 | 3 | 约 2-3 年 |
| 7-8 | 4 | 约 3-4 年 |
| 9-10 | 5 | 约 4-5 年 |
| ... | ... | 每 2 次 +1，无上限 |

### 2.3 概率计算

```
全池总机会数 = (POOL_SIZE - 1) × avg_others_chances + 我的基数
单次命中率    = 我的基数 / 全池总机会数
本期中签概率  = 1 - (1 - 单次命中率) ^ QUOTA_PER_DRAW
```

初始概率示例（基数=1，others_avg=7）：
- 全池总机会数 ≈ (2,600,000 - 1) × 7 + 1 ≈ 18,200,000
- 单次命中率 ≈ 1 / 18,200,000 ≈ 0.0000055%
- 每期抽 19,000 次：P ≈ 1 - (1 - 0.0000055%)^19,000 ≈ **0.1043%**

### 2.4 中签判定

```python
won = random.random() < (probability_pct / 100)
```

### 2.5 模拟流程

```
申请（基数=1）
  ↓
第1期摇号 → 未中 → 基数=1
  ↓
第2期摇号 → 未中 → 基数=2（每2期+1）
  ↓
第3期摇号 → 未中 → 基数=2
  ↓
...
  ↓
第N期摇号 → 中签 🎉
  ↓
退出摇号池（后续摇号被阻止）
```

## 三、数据结构

```python
LotteryProfile:
  name: str               # 申请人姓名（默认"我"）
  total_attempts: int     # 累计摇号次数
  is_winner: bool         # 是否已中签
  won_at_attempt: int | None  # 第几期中签
  records: list[DrawResult]   # 摇号记录列表

DrawResult:
  attempt: int            # 第几期（1-indexed）
  won: bool               # 本期中签？
  chances: int            # 本期基数
  total_pool_chances: int # 全池总机会数
  pool_size: int          # 池子人数
  quota_per_draw: int     # 本期配额
  probability_pct: float  # 本期概率（保留4位小数）
```

## 五、技术决策

### 5.1 内存存储而非数据库

- 单人模拟，数据量小（最多几百条记录），无需持久化
- 重启服务即重置，符合"模拟器"定位
- 如需持久化，加 SQLite 只需一个文件

### 5.2 纯概率模拟而非实际摇号算法

- 实际北京摇号使用"随机抽取"算法（从池中抽 N 个编码）
- 本模拟器简化为：计算个人概率 + random 判定
- 数学上等价（大数定律下），且不需要穷举全池

### 5.3 一年两期而非每月

- 北京 2021 年后燃油车实际为每年 6月、12月两次
- 快进单位使用"年"而非"月"，更符合直觉

## 七、已知限制

1. **家庭摇号未覆盖** — 北京 2021 年后大部分指标通过家庭摇号分配，个人中签率实际上更低
2. **池子规模固定** — 实际池子人数逐期变化（新加入 + 中签退出），本模拟器用固定值估算
3. **他人基数固定** — 假设全池平均摇号 12 次（基数 7），实际分布不均
4. **无跨期关联** — real 系统中签退出后池子变化影响其他申请人，本模拟器不考虑
5. **概率近似** — 简化公式与真实摇号在大样本下一致，但单次可能有细微偏差
