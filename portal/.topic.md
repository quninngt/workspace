## 2026-05-26: 项目创建

- 统一管理所有运行中 Web 服务的控制面板
- FastAPI 后端（8002）+ React 前端（5175）+ Cloudflare 隧道
- 前台有 vite proxy，`/api` 代理到 8002

### 架构决策

- **端口检测用 TCP 而非 HTTP health check** — 通用，无需每个服务配合
- **停止用 SIGKILL（fuser/lsof）而非 SIGTERM** — dev server 无优雅关闭必要
- **启动用 subprocess.Popen 非阻塞** — 不等待服务就绪，立即返回
- **三态模型**：running（前后端均在线）/ partial / stopped
- **Portal 自身卡片隐藏操作按钮** — 防止误关管理面板
- **8 秒自动轮询**，操作后 1.5 秒主动刷新

### 添加新服务

在 `services.py` 的 `SERVICES` 数组中新增一项即自动出现在面板。
